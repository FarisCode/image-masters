# Image Optimizer - Professional WebP Conversion & Optimization

A professional-grade web application for optimizing images and converting them to modern formats like WebP. Built with vanilla JavaScript, HTML5, and CSS3 for optimal performance.

## Features

### Core Functionality
- **Multiple Format Support**: Convert between WebP, JPEG, and PNG formats
- **Batch Processing**: Upload and process multiple images simultaneously
- **Quality Control**: Adjust compression quality from 1-100%
- **Image Resizing**: Set maximum width and height constraints
- **Drag & Drop**: Intuitive drag-and-drop interface
- **Real-time Preview**: See images before processing

### Optimization
- **WebP Conversion**: Convert images to WebP for superior compression
- **Size Reduction**: Automatic optimization for web delivery
- **Dimension Control**: Resize images while maintaining aspect ratio
- **Batch Downloads**: Download all optimized images at once

### Professional UI
- Clean, minimal design with professional aesthetics
- Responsive layout for all device sizes
- Accessibility-first approach
- Smooth animations and transitions
- Dark mode friendly (system preference aware)

## Usage

### Getting Started
1. Open `index.html` in a modern web browser
2. Drag and drop images or click to browse
3. Adjust optimization settings:
   - Select output format (WebP, JPEG, or PNG)
   - Set quality level (1-100%)
   - Optionally set max width/height
4. Click "Process Images"
5. Download individual images or all at once

### Supported Formats
- **Input**: JPEG, PNG, WebP
- **Output**: WebP, JPEG, PNG

## Best Practices

### For Web Optimization
1. Use WebP format for modern browsers
2. Set quality to 80-85% for optimal size/quality ratio
3. Resize large images to appropriate dimensions
4. Process multiple images in batches

### Quality Settings
- **90-100%**: Minimal compression, highest quality
- **80-89%**: Recommended for most web images
- **60-79%**: Good for thumbnails and previews
- **1-59%**: Maximum compression, use carefully
